Castle run tutorial

In this tutorial we will program the gamecastle run.

The game works so that the background moves indefinitely.
The knight (player) must avoid the arrows, either the arrow jumps or makes a somersault and the arrow flies over him as you can see in the video below.

*video contains audio

The player controls the knight using the S keys (makes a somersault) and the space bar (jumps). The game is gradually accelerating.
You can turn the music on or off.
